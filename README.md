# Expense-management
Expense Tracker Application
It is a single page web app that helps you calculate and visualise your budget.

Key features
Add or remove income/expense entities as you wish
Automatically calculates percentages of your total and individual expenses
Stores data in local-storage of your browser, so you don't lose it even if your device shuts down unexpectedly
Clean and easy to use UI
Languages
HTML, CSS
Javascript (General ES 5 syntax used, so no need of any other tool to build)
